Domain

Research Domain

A deep dive into AI-assisted education for Sri Lankan O-Level Mathematics students.

📚

Literature Survey

Existing AI-based educational platforms globally lack alignment with the Sri Lankan O/L Maths syllabus. No Sinhala-medium AI system exists for this domain. Current tools lack model paper generation, step-by-step AI guidance, and stress detection.

🔍

Research Gap

No platform provides AI-generated model papers/quizzes, step-by-step problem explanations, integrated progress tracking, and stress detection — all aligned to the Sri Lankan syllabus in a single system.

⚠️

Research Problem

O/L Mathematics students struggle with past papers, lack personalized guidance, and experience high exam stress. Teachers cannot individually monitor each student's progress effectively.

🎯

Research Objectives

Develop an AI-powered platform for O/L Maths supporting: practice paper generation, AI-guided problem solving, real-time progress tracking, and emotion-based stress monitoring with explainable AI.

🔬

Methodology

RAG (Retrieval Augmented Generation) for math problem solving, computer vision for stress/emotion detection via webcam, ML-based progress analytics, and peer-to-peer forum integration.

⚡

Technologies Used

Python · Flask · React · AI/ML Models · Computer Vision · RAG Architecture · REST APIs · Emotion Detection · Explainable AI (XAI)



Key Features

What the Platform Delivers

01 / COMPONENT

Model Paper & Quiz Generation

AI-generated model papers with short-answer, structured, essay, and full-paper modes. Aligned to the Sri Lankan O/L Mathematics syllabus.

02 / COMPONENT

Peer-to-Peer Forum

An AI-powered mathematics discussion forum where students can ask questions only related to maths, and the system provides answers in Sinhala.

03 / COMPONENT

Stress Detection System

Real-time webcam-based emotion detection during exam sessions. Motivational responses triggered by detected stress states.

04 / COMPONENT

Progress Tracker & Achievements

Explainable AI-powered progress feedback. Tracks performance across topics and provides actionable improvement insights.

05 / IMPACT

Commercialization Plan

Pilot with select schools → full rollout → continuous feedback loop. Low-cost online platform for schools and individual students across Sri Lanka.
